About the Project

This project aims to addressthe communication barriers for both teachers and learners during the learning process as a mechanism of promoting innovation. It will provide a platform in an inclussive setting for accessing learning materials in a visual format thus making the curriculum content more accessible to all school going learners.

This will be achieved through the use of interpretation of static Ugandan Sign language to text, using deep learning.
